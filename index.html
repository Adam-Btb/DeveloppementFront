Bilan Global du Module

Synthèse des acquis

Durant ce module de Développement Front, j'ai acquis des compétences variées :

  • Structure et Style : Maîtrise du HTML5 et du CSS3 pour le design.
  • Mise en page moderne : Utilisation de Flexbox et Grid pour des structures complexes.
  • Responsive Design : Adaptation aux mobiles et tablettes via les Media Queries.
  • Animations : Création d'interfaces dynamiques (transitions, keyframes).
  • Interactivité : Bases de JavaScript (DOM, événements) et utilisation de l'élément <canvas>.

Améliorations futures

Pour aller plus loin, voici mes objectifs :

  • Frameworks JS : Continuer à découvrir le React.js
  • Backend : Lier le site à une base de données.
  • Design System : Utiliser des bibliothèques comme Bootstrap ou Tailwind.
