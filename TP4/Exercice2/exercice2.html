Une liste de documents à télécharger

Bilan personnel

Connaissances : J'ai appris à utiliser les "sélecteurs d'attributs" en CSS. J'ai compris comment cibler un lien selon la fin de son adresse (avec "$=") pour afficher automatiquement une icône PDF ou Word. J'ai aussi utilisé la fonction "attr()" pour afficher le nom du fichier directement via le CSS.

Difficulté : Niveau moyen. La syntaxe est un peu particulière avec les crochets, mais c'est logique une fois qu'on a compris le principe.

Avis : C'est une technique très puissante. Elle permet de garder un code HTML très propre (sans ajouter de classes partout) et de laisser le CSS gérer toute la décoration intelligemment.

Résultat : Le résultat est top : les icônes et les drapeaux de langue s'affichent tout seuls à côté des liens, sans que j'aie besoin de toucher au HTML.
